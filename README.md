# Huddle — Review Site (GitHub Pages ready)

## Deploy
1. Push this folder's contents to the repo (root or /docs).
2. Repo Settings → Pages → deploy from branch → select root (or /docs).
3. Site serves at https://<user>.github.io/<repo>/ — index.html is the landing page.

## Structure
- index.html — This is Huddle (landing)
- the-system.html — The System (three pillars)
- about.html — About
- assets/ — extracted videos + favicon (must deploy alongside the HTML)
